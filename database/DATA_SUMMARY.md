## 数据特点

### 1. 真实性
- 所有地震数据的地理位置均基于真实的中国地震活跃带
- 经纬度坐标与实际地理位置匹配
- 时间分布覆盖2021-2025年，符合历史地震记录规律

### 2. 多样性
- **时间维度**：跨越多个年份和月份，便于时间趋势分析
- **空间维度**：覆盖中国7个省级行政区的30+个县市
- **数据源维度**：包含业务报送和泛在感知两类来源
- **载体维度**：文字、图像、音频、视频四种类型均衡分布

### 3. 完整性
所有数据记录包含完整的36位一体化编码，编码结构：
```
地理信息码(12位) + 时间码(14位) + 来源码(3位) + 载体码(1位) + 灾情码(6位)
```
